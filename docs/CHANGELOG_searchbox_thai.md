# Search Box Placeholders: Translated to Thai

**Date:** 2026-03-11

## Changes Made

### `index.html`

Translated the three remaining English `placeholder` attributes in the search inputs to Thai:

| Tab | Before | After |
|---|---|---|
| Products (`#search-input`) | `Search by part code, name, or SKUs...` | `ค้นหาด้วยรหัสสินค้า ชื่อ หรือ SKU...` |
| Stock Moves (`#moves-search-input`) | `Search by part code or name...` | `ค้นหาด้วยรหัสสินค้าหรือชื่อ...` |
| Flags (`#flags-search-input`) | `Search flagged products...` | `ค้นหารายการของไม่ตรงหน้าเครื่อง...` |

The Customer and Invoice search boxes were already in Thai and were not modified.
